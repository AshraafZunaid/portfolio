# Azure Platform Portfolio

A premium single-page portfolio for GitHub Pages built with React, Vite, Tailwind CSS, Framer Motion, and Lucide icons.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages setup

1. Create a GitHub repository.
2. Push this project to the repository.
3. Update the `base` value in `vite.config.js` to match your repository name if needed.
4. Enable GitHub Pages to deploy from GitHub Actions.
5. Commit the included workflow.

## Customize

- Replace placeholder GitHub and LinkedIn links in `src/App.jsx`.
- Add your resume PDF under `public/` and link it in the hero/contact sections if you want a direct download.
- If your repo name changes, update `base` in `vite.config.js`.
